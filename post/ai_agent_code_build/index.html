[!WARNING]

  • MacのsudoはLinuxとほぼ同じだが、一部コマンドの挙動が異なる場合あり（eg; パッケージ管理はaptではなくbrewなど）
  • システム保護のため、/System ディレクトリなどは特別な権限が必要な場合あり（SIP = System Integrity Protectionの影響）
  • 実行権限の有無確認 
    $ sudo -v
    • [失敗]: mac-user is not in the sudoers file. のようなエラーが出たら権限不足
