Tableau 拡張子:


.twbx を解凍すると、中に .twb やデータファイルが含まれていることを確認できる。
Tableau Server/Online にアップロードする場合、どちらの形式でも可能だが、.twbx が一般的。

拡張子の比較

拡張子 正式名称 内容 データの有無 ファイル形式 主な用途
.twb Tableau Workbook ワークブックの設計情報（ダッシュボード、計算式、接続設定など）を含む ❌ 含まない XML（単一） データソースを別管理する開発用。ファイルサイズが小さい。
.twbx Tableau Packaged Workbook .twb + ローカルデータ（Excel/CSV等）や画像リソースをパッケージ化したファイル ✅ 含む ZIP（圧縮） 配布用。データソース不要で全てを1ファイルで共有可能（例：Tableau Public）。
.tds Tableau Data Source データソースの接続情報（フィールド設定、計算式、接続方法など）を含む ❌ 含まない XML（単一） 同じデータソースを複数のワークブックで再利用する場合に便利。
.tde
.hyper		 Tableau Data Extract 高性能なデータ抽出ファイル（圧縮・最適化されたデータを含む） ✅ 含む 独自バイナリ 大規模データの高速化やオフライン分析用。データ更新には再抽出が必要。

補足説明

  • .twb vs .twbx
    • データを一緒に渡す必要がある場合は .twbx、データソースが別途ある場合は .twb を選択。
  • .tds の利点
    • 同じデータソースを複数のワークブックで共有する場合、接続設定を一括管理可能。
  • .tde の特徴
    • データを圧縮・集約するため、パフォーマンス向上に役立つが、更新時は再作成が必要。

勘所

  • 開発時
    • .twb（軽量） + .tds（データソース共通化） + .tde（データ高速化）
  • 配布時
    • .twbx（すべてをパッケージ化）

コマンド操作

# ダウンロードしたファイルを確認
% cd your/tableau_project/
% ls -l
shop_data_tableau_workbook.twbx

# .twbx を .zip にリネーム
% mv shop_data_tableau_workbook.twbx shop_data_tableau_workbook.zip

# .zip 展開して、展開したファイルを確認
% unzip shop_data_tableau_workbook.zip
% cd shop_data_tableau_workbook/
% ls -l
data/ shop_data_tableau_workbook.twb

# 閲覧
